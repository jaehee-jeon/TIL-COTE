## 문제 2 - 배열에서 문자열 대소문자 변환하기 
### ❌ TRY
```python
def solution(strArr):
    answer = []
    
    for i in strArr :
        if range(1, len(strArr), 2) :
            answer.append(i.upper())
        else :
            answer.append(i.lower())
    
    return answer
```
- 오류 :
    - 조건문의 참/거짓 판단하는데 빈 값이 아니라 참으로 ㅎ

### ✅ ANSWER
```python
def solution(strArr):
    answer = []
    # idx는 0부터 시작하는 인덱스, val은 해당 인덱스의 값
    for idx, val in enumerate(strArr):
        if idx % 2 == 1:  # 홀수 번째 인덱스
            answer.append(val.upper())
        else:             # 짝수 번째 인덱스 (0 포함)
            answer.append(val.lower())
    return answer
```
- 배운 점:
    - enumerate() : 파이썬에서 리스트의 인덱스와 값을 동시에 다룰 때
        - (인덱스, 요소)로 처리함

## 문제 3 - 접두사인지 확인하기
### ❌ TRY
```python
def solution(my_string, is_prefix):
    answer = 0
    
    test = []
    test = my_string[:]
    
    if is_prefix in test :
        return 1
    
    return answer
```
- 오류 :
    - my_string[:]
        - 내가 예상 한 값은 만약에 my_string이 banana이면...
        - b ba ban bana banan banana 이런식으로 될거라 생각했는데..ㅎ
        - banana 로 그대로 나온다고 함 ㅎ
        - > 전체를 그대로 복사하는 것과 같다고 함!   

### ✅ ANSWER 1 (슬라이싱)
```python
def solution(my_string, is_prefix):
    # 접두사의 길이만큼 원본의 앞부분을 잘라서 직접 비교
    if my_string[:len(is_prefix)] == is_prefix:
        return 1
    return 0
```

### ✅ ANSWER 2 (startswith())
```python
def solution(my_string, is_prefix):
    # 접두사로 시작하면 True(1), 아니면 False(0) 반환
    return 1 if my_string.startswith(is_prefix) else 0
```
- 배운 점:
    - startswith() : 접두사 확인하는 전용 메소드
