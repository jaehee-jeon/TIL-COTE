## 문제 1 - 홀짝에 따라 다른 값 반환하기
### ❌ TRY
```python
def solution(n):
    answer = 0
    tmp = []
    if n%2==0 :
        #짝수라면 n이하의 짝수인 모든 양의 정수의 제곱의 합
        tmp = range(2:n:2)
        answer = sum(tmp**2)
    else :
        tmp = range(1:n:2)
        answer = sum(tmp)
    
    return answer
```
- 오류 :
    - *TypeError: unsupported operand type(s) for ** or pow(): 'range' and 'int'
        - 모든 양의 정수의 제곱의 합 : list 배열이므로 불가 
    - range(1:n:2) => 마지막 숫자가 나오지 않음 range(1:n+1:2) 정정 필요
    - range(start, stop, step) 가 맞는 문법

### ✅ ANSWER
```python
def solution(n):
    if n % 2 != 0:
        return sum(range(1, n + 1, 2))
    
    return sum(i**2 for i in range(2, n + 1, 2))
```
- 배운 점:
    - range(start, stop, step)
      - range(start, stop+1, step) 해야 원하는 답까지 도출
    - 제너레이터 표현식    
        - (i**2 for i in range(2, n + 1, 2)

## 문제 3 - 접미사 배열
### ❌ TRY
```python
def solution(my_string):
    answer = []
    
    # 리스트가 my_string 길이 만큼 나오므로.. len(my_string)
    # 배열을 abc 순으로 : sorted 
    answer = sorted[i for _ len(my_string)]
    
    return answer
```
- 오류 :
    - sorted()가 맞다. 함수임..

### ✅ ANSWER
```python
def solution(my_string):
    answer = sorted([my_string[i:] for i in range(len(my_string))])
    
    return answer
```
- 배운 점:
    - for i in range(len(my_string)) 
        - ✨range(len(my_string)) 인가?
            - for i in len(my_string)를 하게되면 for문은 6에서 꺼낼 것이 없어서 에러 발생
        - ✨range(len(my_string))로 하게 되면..
            - range(6)가 0, 1, 2, 3, 4, 5라는 숫자들이 담긴 나열된 객체 만듦...!
    - [:] 슬라이싱을 쓰는 것이 더 효율적
