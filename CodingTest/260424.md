## 문제 1 - 배열의 원소만큼 추가하기
### ✅ ANSWER 
```python
def solution(arr):
    answer = []
    
    for x in arr :
        for _ in range(x) :
            answer.append(x)
    
    return answer
```
- 배운 점:
    - append() : 요소 추가

## 문제 2 - rny_string
### ✅ ANSWER
```python
def solution(rny_string):

    return "".join(['rn' if i == 'm' else i for i in rny_string])
```
- 배운 점:
    - 리스트 컴프리헨션은 두 가지가 있다.
        - ① 필터링 (조건에 맞는 것만 골라낼 때)
            - [결과값 for 변수 in 반복가능객체 if 조건문]
        - ② 변환 (모든 요소를 검사해서 값을 바꿀 때) ✔️
            - [결과값 if 조건문 else 다른값 for 변수 in 반복가능객체]
