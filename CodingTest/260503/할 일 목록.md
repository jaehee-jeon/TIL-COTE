### ✅ ANSWER
```python
def solution(todo_list, finished):
    answer = []

    answer = [todo for i, todo in zip(finished, todo_list) if not i]
    
    return answer
```
- 배운 점:
    - 동시 순회*
        - for a, b in zip(arr, brr)
        - *zip : 두 리스트의 원소를 짝지어(튜플) 하나씩 꺼내줌
    - if i == false 보다 if not i가 더 파이썬스러움.
