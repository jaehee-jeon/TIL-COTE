## 문제 1 - 가까운 1찾기(재도전 🔄)
### ❌ TRY
```python
def solution(arr, idx):
    # 없을 경우
    answer = -1
    
    # 인덱스 값도 필요하니.. enumerate로..
    for i, n in enumerate(arr) :
        # 인덱스 값이 idx보다 크거나 값고 n이 1이면
        if idx <= i and n == 1 :
            # 해당 i값을 ...
            answer = i
    
    return answer
```

- 오류 :
    - 최소값 갱신 : for문을 돌면서 가장 큰 값을 나타냄
    - 비효율적인 탐색 : 가장 작은 값을 만족하는 경우 return으로 바로...
    - * index() 메서드 사용이 더 용도에 맞음

### ✅ ANSWER (index)
```python
def solution(arr, idx):
    
    try :
        return arr.index(1,idx)
    except ValueError:
        
        return -1
```
- 배운 점:
    - index()
        - arr[idx:]에서 1의 위치를 찾고, 시작 오프셋 idx를 더함

## 문제 2 - 주사위 게임 2
### ✅ ANSWER (Set) 
```python
def solution(a, b, c):
    answer = 0
    
    # set : {}를 통해 중복되는 것을 없앰
    cnt = len({a,b,c})
    
    # 하나도 동일한 게 없는 경우
    if cnt == 3 :
        answer = a + b + c
    # 두 개가 동일한 경우
    elif cnt == 2 :
        answer =  (a + b + c) * (a**2 + b**2 + c**2 )
    # 세 개 모두 동일한 경우
    else :
        answer = (a + b + c) * (a**2 + b**2 + c**2 ) * (a**3 + b**3 + c**3 )
    
    return answer
```
- 배운 점:
    - Set (값만 존재)
        - 용도 : 중복을 허용하지 않는다

## 문제 3 - 수 조작하기 1
### ✅ ANSWER (dictionary)
```python
def solution(n, control):
    answer = 0
   
    # 딕셔너리로 정의
    mv = {'w':1, 's':-1, 'd':10, 'a': -10}
    
    for i in control :
        n += mv[i]

    return n
```
- 배운 점:
    - Dictionary (Key:Value 존재)
        - 매핑/조회용
